Place PNG images for each planet symbol in this folder. Example filenames:

- mercury.png
- venus.png
- earth.png
- mars.png
- jupiter.png
- saturn.png
- uranus.png
- neptune.png
- sun.png

Each image should be a small, transparent PNG (e.g., 32x32 or 48x48 px).

After adding the images, update pubspec.yaml to include these assets.
